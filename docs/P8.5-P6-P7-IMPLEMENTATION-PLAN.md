# Part B — P8.5-P7 Communication Intelligence (Whisper Flow)

## B.1 Goal — Effortless Voice Dictation

**Core Wispr feeling (this is P7 — not optional):**

```text
Speak
  → Correct mistakes (heuristics + P6 name memory)
  → Type into focused field
```

**Not** this:

```text
Speak → ChatGPT writes a beautiful essay → paste
```

Creative AI/GPT rewriting (“make this a beautiful email”) remains a **bonus**.
Structured correction classification is now required for ambiguous speech; it
fails open to the literal transcript and never blocks typing.

**Not** OS automation. **Not** semantic agent (“check my code”). Dual-mode with Command unchanged.

| Hotkey | Mode | Behavior |
|--------|------|----------|
| **Ctrl+Space** | Command | Planner + tools (open, audit, analyze…) |
| **Alt+Space** | Dictation / Whisper Flow | Buffer → **correct** → **smart insert** — **never planner** |
| **Escape** | Cancel | Abort buffer / voice |

Overlay must show mode: **Command** vs **Dictation**.

### End-to-end target pipeline (all of P7)

```text
Microphone
  → Whisper STT
  → Dictation mode gate (Alt+Space only for this path)
  → Revision buffer (dictationSession)
  → Correction engine (REQUIRED): "no no", remove, light cleanup
  → P6 voice corrections (REQUIRED for P7.4): nor → Noor
  → Structured correction classifier (ambiguous signals only; literal fail-open)
  → Optional rewrite generator (tone/rewrite decisions only)
  → Confirm final text
  → Focus capture + restore
  → P7.3 Smart typing ladder (OS-first)  ★ ship gate
       1. native SendInput text
       2. sendkeys
       3. clipboard Ctrl+A / Ctrl+V
       4. extension replace (WhatsApp optional fallback only)
       5. vision (optional / last)
  → Verify value grew / matches (not a11y placeholder name)
  → Done
```

**Primary Wispr demo = Alt+Space.** Command+WhatsApp override may reuse the same insert ladder, but must not be the only path.

### What P7 completion gives you (after P7.3 + P7.4)

This is genuinely **Wispr Flow territory**:

| Capability | Example |
|------------|---------|
| **Voice typing** | Alt+Space → `hello john can you send me the report tomorrow` → types into focus |
| **Light cleanup** (optional heuristic) | Same line may capitalize lightly — **not** essay rewrite |
| **Personal corrections** | `nor please check this` → `Noor, please check this.` |
| **Inline editing** | `no no change tomorrow to Friday` → …report **Friday**? |
| **Multi-app typing** | WhatsApp · Gmail · Cursor · Notepad |

Ship bar: those four surfaces type reliably. GPT polish can come later as bonus.
