## READ THIS FIRST — 4 different “find/open” systems

| You say | System | Route | Prerequisite |
|---------|--------|-------|--------------|
| `Open last pdf` / `Open last folder I opened` | **Session recall** | `recall_memory` (legacy desktop path) | You must have opened a pdf/folder earlier **this session** |
| `Open yesterday's pdf` / `Open today's pdf` | **Temporal smart search** | `memory.search` (P8.5 tool) | File index / retriever has matching files |
| `Search my resume` / `open my resume` | **Name/token search** | `memory.search` | Indexed files matching token |
| `Open my portfolio` (after you taught it) | **User alias** | `open_alias` → opens saved path | Run `Remember …` first |
| `open downloads` | **Well-known folder** | `filesystem.open` | None |

**Do not confuse:**
- `Open last pdf` ≠ `Open yesterday's pdf`
- `Move Invoice.pdf from Downloads to Desktop` (file) ≠ `move cursor down` (keyboard)
